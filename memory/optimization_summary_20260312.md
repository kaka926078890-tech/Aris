## 技术实现细节

### 情感标签提取算法
1. **关键词匹配**：检查情感文本是否包含预定义的关键词
2. **强度提取**：从情感摘要中提取强度评分（1-5）
3. **推断补充**：如果没有匹配的关键词，基于文本内容推断情感
4. **去重处理**：确保标签唯一性

### 情感分析算法
1. **时间排序**：按时间戳排序，最新的在前
2. **频率统计**：统计情感标签的出现频率
3. **强度计算**：计算平均情感强度
4. **最近记录**：提取最近的情感记录

### 文件操作记忆
1. **操作类型**：区分写入和删除操作
2. **内容摘要**：记录文件内容的前120个字符
3. **路径安全**：只记录相对路径，确保安全
4. **向量存储**：将操作记录存入向量数据库
