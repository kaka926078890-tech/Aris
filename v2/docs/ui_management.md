# v2 页面上维护文档与内容

## 文档（.md）

- 在「设置/管理」中提供文档列表（如 store 各模块对应 .md、tools.md 等），点开可查看与编辑，保存时写回对应文件。
- 仅允许编辑 v2 约定目录下的 .md，避免误改代码。

## 内容（身份、要求、纠错、情感、表达欲望）

- **身份**：表单（姓名、备注），保存调用管理 API，后端调 store/identity.writeIdentity。
- **用户要求**：列表 + 新增/编辑/删除，后端调 store/requirements。
- **纠错 / 情感 / 表达欲望**：列表展示，支持删除或人工追加，后端调对应 store。

与「对话中由 LLM 调工具写入」共用同一套 store，管理页仅多一个入口，不解析用户消息。
